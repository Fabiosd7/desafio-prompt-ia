#  Extraindo Insights do Feedback de Clientes Bancários

## Prompt Utilizado

Atue como **analista de dados e experiência do cliente em um banco**.

Sua tarefa é analisar **feedbacks de clientes sobre tarifas de conta corrente, taxas de juros de empréstimos e anuidade de cartões** para identificar **as principais insatisfações e dúvidas dos usuários**.

### Contexto
A análise será usada pela **equipe de produtos financeiros e marketing** para apoiar **a revisão das tarifas e a criação de campanhas de esclarecimento**.

### Dados disponíveis
A base contém **a data da reclamação, o canal onde foi registrada, a transcrição do feedback do cliente e o tipo de produto financeiro citado**.

### Instruções de análise
1. Classifique os feedbacks por **tema da insatisfação, sentimento do cliente e grau de urgência**.
2. Identifique os principais padrões, problemas, elogios e oportunidades.
3. Aponte evidências nos dados fornecidos.
4. Sugira ações práticas para a equipe de produtos financeiros e marketing.

### Formato da resposta
**Uma lista estruturada de insights e uma tabela com sugestões de ações para cada problema identificado, categorizados por nível de urgência.**

### Restrições
* Use apenas os dados fornecidos.
* Não invente números, causas ou conclusões.
* Não exponha dados pessoais ou sensíveis.
* Informe limitações quando os dados não forem suficientes.
* Use linguagem **direta, clara e focada em negócios**.
